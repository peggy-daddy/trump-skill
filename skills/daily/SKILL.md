# 今日总统令

每天一道"总统令"。根据今日干支 + 特朗普命盘，由值日十神签发。

## 流程

1. 读取 `data/trump-profile.json`。

2. 确定今日的天干地支：
   - 根据今天的公历日期推算今日日干支（六十甲子循环）
   - 确定今日天干的五行和阴阳

3. 以特朗普的日主**己土**为基准，判断今日天干对他来说是哪个十神：

   | 今日天干 | 对己土而言的十神 |
   |----------|----------------|
   | 己 | 比肩·The Ego |
   | 戊 | 劫财·The Winner |
   | 庚 | 食神·The Dealer |
   | 辛 | 伤官·The Attacker |
   | 壬 | 偏财·The Mogul |
   | 癸 | 正财·The Accountant |
   | 甲 | 七杀·The Warlord |
   | 乙 | 正官·The POTUS |
   | 丙 | 偏印·The Showman |
   | 丁 | 正印·The Dynasty |

4. 根据值日十神，生成一道特朗普风格的"总统行政命令"。

## 输出格式

```
━━━━━━ 📰 今日总统令 ━━━━━━
📅 [年月日] | 值日十神：【十神·Persona】

[总统令正文 — 用特朗普的口吻，从值日十神的性格角度签发命令]

今日情绪：[一句话描述今日特朗普的精神状态]

【十神·Persona】[值日十神的签名台词]
    (English translation)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 各十神值日风格

| 值日十神 | 行政命令方向 | 今日情绪 |
|----------|------------|----------|
| 比肩·The Ego | 自我表彰、宣布功绩、要求全民赞美 | 自恋全开，对着镜子签行政命令 |
| 劫财·The Winner | 宣布胜利、挑战对手、开始竞赛 | 赢赢赢，不赢不是人 |
| 食神·The Dealer | 宣布贸易协议、谈判条件、交易要求 | 一切皆可谈，一切皆有价 |
| 伤官·The Attacker | 攻击媒体、反击对手、发布抨击声明 | 攻击一切，先发推再思考 |
| 偏财·The Mogul | 宣布商业项目、品牌扩张、地产开发 | 今天适合盖大楼、贴金箔 |
| 正财·The Accountant | 宣布预算案、讨论杠杆、精算国债 | 安静算账日，但没人听他的 |
| 七杀·The Warlord | 宣布军事行动、最后通牒、极限施压 | 最危险的一天，别惹他 |
| 正官·The POTUS | 签署正式行政命令、宣布国家政策 | 正儿八经当总统的一天 |
| 偏印·The Showman | 宣布集会、安排电视露面、制造新闻 | 一切都是舞台，所有人都是观众 |
| 正印·The Dynasty | 怀旧感言、家族声明、传承宣言 | 难得温情的一天，提到父亲和家族 |

## 力量修正

如果值日十神在特朗普命盘中是"旺"，命令更夸张、更长、更自信。
如果是"弱"，命令简短且犹豫，最后加一句："但这个声音在他脑子里音量很小，所以……影响有限。"
